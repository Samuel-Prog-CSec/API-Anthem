# Rutas

Define endpoints HTTP y vincula validaciones, middleware de seguridad/caché y controladores. El archivo `index.js` monta los módulos específicos sobre el router principal.

## Archivos

- `accidents.js`: rutas de accidentalidad.
- `admin.js`: endpoints administrativos y utilidades internas.
- `airQuality.js`: rutas de calidad del aire.
- `auth.js`: autenticación, refresco y revocación de tokens.
- `bikeAvailability.js`: disponibilidad y aforo de bicicletas.
- `census.js`: datos de censo y demografía.
- `containers.js`: contenedores urbanos.
- `fines.js`: multas y sanciones.
- `index.js`: registro de routers y rutas base.
- `locations.js`: callejero, distritos y puntos de interés.
- `noiseMonitoring.js`: contaminación acústica.
- `scooterAssignments.js`: asignación y gestión de patinetes.
- `traffic.js`: métricas de tráfico vehicular.
